\# 📚 BookVerse



BookVerse is a frontend web application built using HTML, CSS, and JavaScript that helps users explore and discover books in an interactive and visually appealing way. It allows users to search, filter, and view books based on genre, rating, popularity, and value.



\---



\## 📝 Abstract



BookVerse is designed to make book discovery simple and engaging. The application allows users to browse a collection of books and analyze them using interactive search and filtering options. Users can search for books by title, filter them by genre and rating, and explore categorized recommendations such as top popular books and best value books.



Each book is displayed in a clean card-based layout with important details like title, author, rating, reviews, price, and genre. The project combines frontend web technologies with data-driven logic to create an attractive and user-friendly platform for book exploration.



\---



\## 📄 Project Description



BookVerse is a web-based frontend application developed using HTML, CSS, and JavaScript. It uses a JavaScript dataset file (`data.js`) that stores book details such as title, author, rating, reviews, price, genre, and year.



The application dynamically displays books in different sections such as:



\- \*\*Top 10 Popular Books\*\*

\- \*\*Best Value Books\*\*

\- \*\*All Books\*\*



It includes a search bar and filter options that allow users to quickly find books based on their interests. Users can also click on a book card to view more detailed information in a popup modal.



The project focuses on modern UI/UX design by using responsive layouts, card-based sections, hover effects, and interactive features to improve the overall user experience.



\---



\## 🎯 Problem Statement



Choosing the right book from a large collection can be difficult for users because of the many available options. Users may struggle to find books that match their interests, quality expectations, and budget. There is a need for a simple and interactive system that helps users explore books efficiently through search, filtering, and categorized recommendations.

\## ⚙️ How It Works



The application loads book data from the `data.js` file and dynamically renders the books using JavaScript. It processes the dataset to calculate popularity and value-based categories, then displays them in separate sections. Users can interact with the interface using the search bar and filters to refine results. Clicking on a book card opens a popup modal with detailed information about the selected book.
